# FlexPro

FlexPro is a Flutter app for FAST NUCES students that makes the Flex experience easier to use on mobile. It brings the most important academic tools into one place: dashboard, marks, attendance, transcript, GPA calculator, faculty directory, and past papers.

## What FlexPro Does

- Dashboard for a quick overview of your academic data.
- Marks with detailed course breakdowns.
- Attendance tracking for each course.
- Transcript view with semester-wise results.
- GPA calculator based on transcript data.
- Faculty directory with teacher profiles and reviews.
- Past papers browser to help with exam preparation.

## Highlights

- Clean mobile-first UI with fast navigation.
- Secure sign-in and session persistence.
- Saved academic data for easier reuse.
- Built with Flutter, Riverpod, GoRouter, Dio, and Hive.
- Release-friendly structure for sharing public builds.
